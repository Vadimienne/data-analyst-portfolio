## 🚚 Задача 2: Влияние доставки на отзывы

**Бизнес-вопрос:**  
*«Как время доставки влияет на рейтинг? Есть ли порог, после которого клиенты массово ставят 1–2 звезды?»*

### 📊 Визуализация
![Delivery vs Rating](./outputs/charts/delivery_binned_rating.png)

### ✅ Ключевые инсайты
| Интервал доставки | Средний рейтинг | Доля заказов |
|------------------|----------------|--------------|
| 0–3 дня | 4.32⭐ | 15% |
| 4–7 дней | 4.18⭐ | 35% |
| 8–12 дней | 3.95⭐ | 27% |
| **13–20 дней** | **3.42⭐** 🔴 | 16% |
| 21+ дней | 2.51⭐ 🔴 | 7% |

> 💡 **Инсайт**: При доставке **>12 дней** средний рейтинг падает с 4.2 до 3.4 (−19%).

### 🎯 Рекомендация
Установить целевой SLA **12 дней**. Для регионов с текущей доставкой >15 дней запустить пилот с локальными складами.

## 📊 Задача 6: Executive Summary *(в работе)*

**Бизнес-вопрос:**  
*«Как выглядит бизнес в одном экране? Какие 3 рекомендации дать руководству?»*

### 🎯 Топ-3 рекомендации для руководства

1. **🚚 Оптимизировать доставку в северных регионах**  
   Снижение времени доставки с 28 до 15 дней в штатах AM/PA может повысить рейтинг с 3.2⭐ до 4.0⭐ → +£180K к годовой выручке.

2. **🎯 Фокус на удержании «Champions»**  
   12% клиентов приносят 45% выручки. Запуск персональной программы лояльности для этого сегмента может увеличить их LTV на 20–30%.

3. **✂️ Сократить «длинный хвост» ассортимента**  
   Удаление 5,810 C-товаров (5% выручки) снизит складские издержки на ~£250K/год без значимого влияния на общую выручку.

| Метрика | Значение | Бенчмарк | Статус |
|---------|----------|----------|--------|
| Средний чек (AOV) | £85 | — | 🟢 |
| LTV_3m | £105 | >£100 | 🟢 |
| Repeat Rate | 14% | >20% | 🟡 |
| Доля доставок >12 дней | 23% | <10% | 🔴 |
