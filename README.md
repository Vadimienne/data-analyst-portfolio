## 📊 Полные итоги

**Бизнес-вопрос:**  
*«Как выглядит бизнес в одном экране? Какие 3 рекомендации дать руководству?»*

### Топ-3 рекомендации для руководства

1. **SLA на доставку <20 дней**
   Сейчас 12% заказов теряют в оценке из-за долгой доставки.
2. **Оптимизировать доставку в северных регионах**  
   Снижение времени доставки в штатах AM/PA может повысить рейтинг 
3. **Сократить «длинный хвост» ассортимента**  
   Удаление 5,810 C-товаров (5% выручки) снизит складские издержки на ~£250K/год без значимого влияния на общую выручку.
